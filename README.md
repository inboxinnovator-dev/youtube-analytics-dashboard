
# YouTube Analytics Dashboard

## Description
A Python-based dashboard that analyzes YouTube Studio CSV data to generate insights such as CTR, watch time, and video performance.

## Problem
Raw YouTube analytics data is difficult to analyze manually.

## Solution
This project uses Python and Pandas to clean data, calculate CTR, visualize performance trends, and generate reports.

## Technologies Used
- Python
- Pandas
- Matplotlib

## How to Run
```bash
pip install -r requirements.txt
python src/dashboard.py
